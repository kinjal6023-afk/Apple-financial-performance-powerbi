# Apple Financial Performance Dashboard (Power BI)

## Overview
This project analyzes **Apple’s financial performance** using a Power BI dashboard built from **2022–2023** financial statement data.  
The report focuses on **liquidity, leverage, and overall financial stability**, presented through interactive visuals and KPIs.

## Dataset
- Source: Public Apple annual financial statements
- Years covered: **2022 and 2023**
- Data format: Clean, long-format Excel data (`Item | Year | Value`)
- Excel file contains **raw financial data only** (no calculations)

## Dashboard Features
- Financial performance overview (Revenue, Assets, Liabilities, Equity)
- Key financial ratios:
  - Current Ratio (liquidity)
  - Debt-to-Equity Ratio (leverage)
- KPI cards built using DAX
- Interactive slicers (Year, Financial Item)
- Business-focused insights for decision-making

## Key Insight (2023 Snapshot)
While overall performance remained stable, **liquidity stayed below ideal levels** and **leverage increased in 2023**, indicating higher short-term financial risk if revenue growth slows.

## Tools & Skills Used
- Power BI (data modeling, DAX, visualization)
- Financial ratio analysis
- Data storytelling
- Excel (data preparation)
